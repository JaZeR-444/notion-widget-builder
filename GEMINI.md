# Notion Widget Builder v5.0

## Project Overview

This project is a web-based visual builder for creating custom widgets for Notion. It allows users to design and export widgets like clocks, countdowns, weather displays, and more. The application is built with React and Vite, and styled with Tailwind CSS.

## Building and Running

### Development
To run the development server:
```bash
npm run dev
```

### Build
To create a production build:
```bash
npm run build
```

### Preview
To preview the production build:
```bash
npm run preview
```

### Linting
To run the linter:
```bash
npm run lint
```

## Development Conventions

The project uses ESLint for code linting. The configuration can be found in `eslint.config.js`. It is recommended to run the linter before committing any changes.
